Outplay Entertainment
Dundee, UK (07/2013 - 08/2015)
Game Engineer

Outplay Entertainment is a mobile games company focused on casual games but that also makes some midcore games. In this company I have been involved in three great projects

Alien Creeps TD I have been in the project during all its process (it was started 2 weeks before I came to the company). This is a Cocos2d-x based game. We were 2 programmers during most of its development, and almost at the end a third one joined us. The team also had 2-3 designers, 2-4 artists and 1-2 producers. This has been the first big game I have been involved, and I had the opportunity to learn a lot and from very (very) talented people, which has allowed me to experience a very fast learning curve.

Since it has been deployed to both iOS and Android, with this project I have finally been given the chance of learning how to structure multiplatform games.

In this project I also made and maintained most of the tools the designers used for creating levels and defining enemies and towers. Those tools were created using Qt Creator.

Bubble Genius In my second project in the company I was the lead programmer. The engineer team was compounded by me and another person for 5 months and a third programmer joined us afterwards. This was a bubble popper game developed using cocos2d-x as well. Here I was able to strengthen my skills applying the correct patterns to the project, as I literally started to code it.

Apart from the game side, I also was responsible of developing the tools for the designers to create the levels. This was done using Qt Creator again.

This was the third and last project I was involved in the company. The project had already been started and I was moved in during the last month to be able to speed up the development of one of the core systems of the game. The engineer team was compound only by another person and me at that time.

iOS developer

At CogniFit I had the opportunity to manage 2 big projects: CogniFit Brain Fitness and CogniFit MoodCraft. My role here was to create the software analysis and implement it. The Brain Fitness app was a port from the website, so the idea was clear, but the design changed several times during the development process. It has a nice editor to create moods (emoticons) and several options to be able to make it social and viral. The programming part of this app was done only by me and took a little bit more than 3 months to launch the first version.
